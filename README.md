# Business Travel Expense Tracker

This is a Java-based app designed to track and manage business travel expenses.

## Features
* Expense and travel logging
* Expense categories
* Approval workflow
* Reimbursement tracking

## Technologies used
* Java
* JavaFX
* Maven
* Google Gson
